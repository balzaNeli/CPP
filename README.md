# 📘 Exercícios de C++ para Aprendizado

Repositório dedicado ao estudo e prática da linguagem **C++**, com exercícios organizados por nível de complexidade.

---

## 📁 Estrutura do Repositório

```
CPP/
├── ExerciciosAvancados/   # Exercícios de nível avançado em C++
├── ExerciciosFaceis/ 
├── utils/                 # Funções e helpers reutilizáveis
├── output/                # Arquivos gerados pela compilação
├── .vscode/               # Configurações do editor VS Code
└── README.md
```

### Descrição das pastas

- **`ExerciciosAvancados/`** — Exercícios mais complexos, abordando tópicos avançados da linguagem como ponteiros, herança, templates, STL, entre outros.
- **`utils/`** — Utilitários e funções auxiliares reutilizáveis que servem de apoio aos exercícios.
- **`output/`** — Diretório onde os binários e arquivos compilados são gerados.
- **`.vscode/`** — Configurações do VS Code (tasks, launch, settings) para facilitar a compilação e debug.

---

## 🚀 Como Compilar e Executar

### Pré-requisitos

- Compilador C++ (`g++` recomendado)
- [VS Code](https://code.visualstudio.com/) com a extensão **C/C++** 

### Compilando manualmente

```bash
g++ -o output/programa ExerciciosAvancados/exercicio.cpp
./output/programa
```

### Usando o VS Code

Com as configurações já presentes em `.vscode/`, basta pressionar `Ctrl + Shift + B` para compilar ou `F5` para compilar e executar com o debugger.

## 🧠 Tópicos Abordados

- Tipos de dados e variáveis
- Estruturas de controle (`if`, `for`, `while`, `switch`)

- Ponteiros e referências
